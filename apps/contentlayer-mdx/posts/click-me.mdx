# Ankit Plugin

Blog posts have their own pages. The content source is a markdown file, parsed to HTML by Contentlayer.

```html
<pre>
  <code>
    <span>export default function Page() {</span>
    <span> return &lt;h1&gt;Hello world!&lt;/h1&gt;</span>
    <span>}</span>
  </code>
</pre>
```
